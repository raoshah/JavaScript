Platform Requirements
Real-Time Metering (RTM) provides the most accurate and real-time usage monitoring. This functionality can be used to track usage and when certain thresholds are met, send API based notifications to the Wholesale Customer and/or automatically cap the usage for that specific device.

Real-Time Data Metering (RTDM) provides the most accurate and real-time data usage monitoring. This functionality can be used to track usage and when certain thresholds are met, send API based notifications to the Wholesale Customer and/or automatically throttle or cap the usage for that specific device. These actions happen automatically with an in-session, real-time, modification of the end user’s data session.

Real-Time Metering (RTM) provides the most accurate and real-time usage monitoring. This functionality can be used to track usage and when certain thresholds are met, send API based notifications to the Wholesale Customer and/or automatically cap the usage or throttle data speed for that specific device. NOTE: For IoTs, only Real time metering of Data is available, for MVNOs, Real time metering for Voice, SMS and Data is available.
